My work

Since you came this far, I hope you're interested in my work.

  • So my first project was called BloodPool which was a Raspberry Pi based automated system to facilitate blood banks in remote areas. This would eliminate the need of having a full fledged management system for blood banks as this device maintains records, publishes demand requests and fulfills needs too. Read more here from page 82.
  • Then we built a smart helmet which would detect accidents and based on location, generate SOS messages. Found this interesting? Have a look here.
  • After this we build a realtime pollution monitoring and green auditing system for various industries in India. This project involved wireless sensor networks talking securely to a central device which sent data to our realtime databse from where we used traditional machine learning to generate a CEPI score. You can access it on GitHub from here
